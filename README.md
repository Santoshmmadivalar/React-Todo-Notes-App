## 🌟 Project Overview

Task Manager & Notes App is a frontend application built using **React.js** and modern JavaScript concepts.

Users can create tasks, track their completion status, and manage personal notes in a clean and interactive interface.

This project demonstrates React fundamentals, reusable components, state management, event handling, and dynamic rendering.

---

## 🚀 Features

### ✅ To-Do Manager

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Real-time task updates
* Responsive UI

### 📒 Notes Manager

* Create notes
* View saved notes
* Delete notes
* Dynamic note rendering
* Clean and user-friendly interface
